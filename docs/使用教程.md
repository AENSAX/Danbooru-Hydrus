# PMTagger 使用教程

本文说明 PMTagger 服务、`PMDanBooruSpider` 和 `PMImageSpider` 的使用方式。

## 1. 准备 Hydrus

1. 打开 Hydrus 客户端。
2. 启用 `Client API`。
3. 创建 Access Key，并至少授予这些权限：
   - `Import Files`
   - `Add Tags`
   - `Search Files`
4. 确认 Hydrus Client API 地址，默认通常是：

```text
http://127.0.0.1:45869
```

## 2. 启动 PMTagger

在 PowerShell 中执行：

```powershell
cd \PMTaggerServer
uv sync
uv run pmtagger-service
```

浏览器打开：

```text
http://127.0.0.1:8000/
```

### 修改启动端口

如果要改端口，例如改成 `8010`：

```powershell
cd \PMTaggerServer
$env:TAGGER_PORT = "8010"
uv run pmtagger-service
```

如果还想允许局域网访问：

```powershell
$env:TAGGER_HOST = "0.0.0.0"
$env:TAGGER_PORT = "8010"
uv run pmtagger-service
```

这时地址会变成：

```text
http://127.0.0.1:8010/
```

在 WebUI 中填写：

- `Hydrus API 地址`
- `Hydrus Access Key`
- `Hydrus 标签服务名`
- 模型和阈值
- 翻译 CSV

然后点击 `保存配置`。

保存后的配置会写入：

```text
\PMTaggerServer\runtime-config.json
```

## 3. PMDanBooruSpider 用法

`PMDanBooruSpider.js` 专用于 Danbooru。

### 安装

1. 安装浏览器扩展 Tampermonkey。
2. 新建脚本。
3. 将 `\PMDanBooruSpider.js` 的全部内容粘贴进去。
4. 保存脚本。

### 配置

打开 Tampermonkey 菜单，执行：

```text
设置 PMTagger 服务地址
```

默认地址是：

```text
http://127.0.0.1:8000
```

可以执行：

```text
测试 PMTagger 服务连接
```

如果提示 Hydrus 未连接，回到 PMTagger WebUI 检查 Hydrus 配置。

### 单图上传

1. 打开 Danbooru 帖子页，例如：

```text
https://danbooru.donmai.us/posts/123456
```

2. 点击右下角 `传送到 PMTagger`。
3. 脚本会读取帖子 JSON、下载原图、收集 Danbooru 标签。
4. PMTagger 会翻译标签，把中文追加到英文后，再上传到 Hydrus。

### 批量上传当前页

1. 打开 Danbooru 列表页，例如：

```text
https://danbooru.donmai.us/posts?tags=1girl
```

2. 点击右下角 `批量导入本页`。
3. 确认后，脚本会逐个处理当前页帖子。
4. 完成后会提示成功数和失败数。

### 标签来源

PMDanBooruSpider 会读取这些 Danbooru 字段：

- 普通标签：`tag_string_general`
- 作者标签：`artist:*`
- 角色标签：`character:*`
- 作品标签：`series:*`
- 物种标签：`species:*`
- 设定标签：`lore:*`
- 元标签：`tag_string_meta`
- 分级标签：`rating:safe|questionable|explicit`
- 帖子标识：`danbooru:<post_id>`

因为脚本已经提供标签，所以 PMTagger 不会再对 Danbooru 图片做 AI 打标，而是直接翻译这些标签并上传。

## 4. PMImageSpider 用法

`PMImageSpider.js` 适用于所有网页。

### 安装

1. 安装浏览器扩展 Tampermonkey。
2. 新建脚本。
3. 将 `\PMImageSpider.js` 的全部内容粘贴进去。
4. 保存脚本。

### 配置

打开 Tampermonkey 菜单，执行：

```text
设置 PMTagger 服务地址
```

默认地址是：

```text
http://127.0.0.1:8000
```

可以执行：

```text
测试 PMTagger 服务连接
```

### 右键图片上传

1. 打开任意网页。
2. 在图片上点击鼠标右键。
3. 页面会出现 `上传图片到 PMTagger` 菜单。
4. 点击后，脚本会下载该图片并发送到 PMTagger。
5. PMTagger 会自动 AI 打标、翻译标签，并上传到 Hydrus。

### 注意事项

- PMImageSpider 不提供手动标签，发送给 PMTagger 的 `tags` 为空。
- 因为 `tags` 为空，PMTagger 会自动执行 AI 打标。
- 某些网站会阻止第三方脚本下载原图，如果上传失败，可以打开浏览器控制台查看具体请求错误。
- 动态加载或懒加载图片通常也支持，只要右键目标是真实的 `<img>` 元素即可。
